# zookeeper 安装

## 开发环境安装
安装docker-compose命令
```bash
$ sudo apt install docker-compose
```
生成docker-compose.yml
```text
version: '2'
services:
    zoo1:
        image: zookeeper
        restart: always
        ports:
            - 192.168.0.14:2181:2181
        environment:
            ZOO_MY_ID: 1
            ZOO_SERVERS: server.1=zoo1:2888:3888 server.2=zoo2:2888:3888 server.3=zoo3:2888:3888

    zoo2:
        image: zookeeper
        restart: always
        ports:
            - 192.168.0.15:2181:2181
        environment:
            ZOO_MY_ID: 2
            ZOO_SERVERS: server.1=zoo1:2888:3888 server.2=zoo2:2888:3888 server.3=zoo3:2888:3888

    zoo3:
        image: zookeeper
        restart: always
        ports:
            - 192.168.0.16:2181:2181
        environment:
            ZOO_MY_ID: 3
            ZOO_SERVERS: server.1=zoo1:2888:3888 server.2=zoo2:2888:3888 server.3=zoo3:2888:3888
```
运行容器
```bash
$ docker-compose up
```
